# api-rusticburguer
Esse projeto tem como foco a construção de uma API REST, utilizando (NodeJS, Sequelize, MySql, Express) de uma Lanchonete de Hamburguer Artesanal, projeto sem fins lucrativos, apenas para Ajudar a fortalecer a lanchonete no comércio local e também portifólio para minha carreira como Desenvolvedor.
